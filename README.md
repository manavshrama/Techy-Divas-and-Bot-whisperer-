# MindMitra GNDEC Edition 🧡🤍💚

Built live in 8-hour hackathon using vibe coding – GNDEC Ludhiana.

A specialized mental health chatbot for Punjab engineering students at GNDEC Ludhiana.
Built with ❤️, Python, Streamlit, and Transformers.

### 🧡 Setup & Installation
Run the following in a Colab cell or your local terminal:
```bash
!pip install streamlit transformers torch plotly pandas speechrecognition pyttsx3 localtunnel
```

### 🤍 Features
- **Emotion Recognition**: Powered by DistilRoBERTa.
- **Bilingual Conversations**: Warm Support in English & Punjabi.
- **Crisis Detection**: Immediate access to Indian mental health helplines.
